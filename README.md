# IS 436 Personal Portfolio Website

This is a Bootstrap-based personal portfolio website created for IS 436. It includes five pages:

- Home
- About Me
- Portfolio / Projects
- Skills & Resume
- Contact

## Features

- Responsive Bootstrap layout
- Professional visual design
- Accessibility support with alt text, keyboard-friendly navigation, and color contrast
- Project content tied to IS 436 learning outcomes
- Ready for GitHub Pages deployment

## How to Publish on GitHub Pages

1. Create a public GitHub repository.
2. Upload all files from this folder.
3. Go to **Settings > Pages**.
4. Under **Branch**, select `main` and `/ (root)`.
5. Save your settings.
6. Your portfolio will appear at:
   `https://your-username.github.io/repository-name/`

## Before You Submit

Update the following details so the portfolio sounds fully personal:

- Add your real headshot if you want
- Replace placeholder LinkedIn and GitHub links
- Update project descriptions if you want them to match your exact coursework
- Add your latest resume link or file
